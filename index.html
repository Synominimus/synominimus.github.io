CSS Topic

In this section we will revise one CSS topic, namely the specificity or hierarchy of selectors.

Meaning
type The type is the lowest tier for selectors
class Classes are used for elements that are recurring throughout the page
id IDs are unique to an element and should only be used for styling unique elements
!important' !important overrides any other selector tier
